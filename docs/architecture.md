# Cocos 3D HMI 配置器 - 架构文档

## 1. 项目概述

**项目名称**: cocos-hmi-configurator  
**项目类型**: 3D产品配置器Demo  
**核心功能**: 3D模型展示、材质切换、360度旋转、配置预览  
**目标用户**: 车企HMI产品展示、电商3D产品预览

---

## 2. 技术架构

### 2.1 技术栈
- **引擎**: Cocos Creator 3.8+
- **语言**: TypeScript
- **渲染**: WebGL 2.0
- **版本管理**: Git

### 2.2 模块划分

```
┌─────────────────────────────────────────┐
│              UI 层                       │
│  (ConfiguratorUI - 按钮、面板)           │
├─────────────────────────────────────────┤
│              业务层                       │
│  (ProductConfigurator - 配置逻辑)         │
├─────────────────────────────────────────┤
│              引擎层                       │
│  (Cocos Core - 渲染、输入、物理)          │
└─────────────────────────────────────────┘
```

---

## 3. 核心模块

### 3.1 ProductConfigurator
| 属性/方法 | 说明 |
|-----------|------|
| productModel | 3D产品模型节点 |
| materials | 材质数组 |
| rotateSpeed | 旋转速度 |
| nextMaterial() | 切换下一材质 |
| setMaterial(index) | 设置指定材质 |
| resetView() | 重置视角 |
| getCurrentConfig() | 获取当前配置 |

### 3.2 ConfiguratorUI
| 属性/方法 | 说明 |
|-----------|------|
| prevBtn/nextBtn | 切换材质按钮 |
| resetBtn | 重置视角按钮 |
| materialLabel | 当前材质显示 |
| showConfirm() | 显示配置确认 |

---

## 4. 交互流程

```
用户触摸 → TOUCH_START → 停止自动旋转 → 手动拖动旋转
        → TOUCH_END → 3秒后恢复自动旋转
```

---

## 5. 材质系统

预置5种材质：
1. 默认白色 - 基础款
2. 炫酷黑色 - 运动款
3. 科技蓝色 - 智能款
4. 活力橙色 - 青春款
5. 奢华金色 - 旗舰款

# Cocos 3D HMI 配置器

基于 Cocos Creator 3D 开发的3D产品配置器Demo。

## 功能特性

- 3D模型展示
- 材质/颜色切换
- 360度旋转查看
- 产品配置预览
- 界面交互

## 技术栈

- Cocos Creator 3.8+
- TypeScript/JavaScript
- 3D渲染

## 项目结构

```
assets/
├── scripts/      # 业务逻辑脚本
├── prefab/       # 预制体
├── scene/       # 场景文件
├── texture/     # 纹理图片
└── model/       # 3D模型
```

## 快速开始

1. 安装 [Cocos Creator](https://www.cocos.com/creator)
2. 克隆本项目
3. 用 Cocos Creator 打开项目目录
4. 点击运行
